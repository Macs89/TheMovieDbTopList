# TheMovieDbTopList
Test project for job application.

**Here is the original task:**

```A feladat egy egyszerű frontend alkalmazás készítése megy a themoviedb.org apira épül majd.
Ehhez szükséges majd egy általd készített API clienttel lekérdezni az adatokat melyekete meg kell majd jeleníteni 
megfelelő módon a felhasználói felületen.

A próbafeladathoz bármely általad kedvelt CSS frameworkot használhatsz (pl. twitter bootstrap, bulm, tailwind stb.) 
illetve bármely javascript frameworköt használhatod (mi VueJs vagy ReactJs rendszereket használunk, de nem probléma 
ha nem ezek egyikében készül). Nem a külalak számít hanem a funkcionális megoldás.

A kész feladatod egy publikus git repositoryba pusholva oszd meg majd velünk kérlek.

Az API dokumentációját az alábbi linken érheted el  https://developers.themoviedb.org/3

A felhasználói felület egy lista nézet legyen mely a TheMovieDb “top rated movies” listájának első 210 helyezettjét 
listázza amit az APIn keresztül kér majd le a rendszer.

A listában megjelenítendő adatok:
helyezés a top listán
title
length
poster url
tmdb vote average
tmbd vote count

A lista legyen rendezhető cím és helyezés alapján, illetve szűrhető cím alapján, valamint 50-esével listázzon oldalanként.

A listaelemekre való kattintáskor egy modalban vagy új oldalon jelenjenek meg a film további részletei valami egyszerű módon 
pl. Táblázatosan. (nem számít  a dizájn megvalósítás). Tehát az alábbi adatokat kell látni itt:
title
length
genre(s)
release date
overview
poster url
tmdb id
tmdb vote average
tmbd vote count
tmdb url
director(s) name

Nem kritérium, de előnyös lenne ha a megoldásod tartalmazna teszteket, hogy biztosítsd a kódod megfelelő működését.```
